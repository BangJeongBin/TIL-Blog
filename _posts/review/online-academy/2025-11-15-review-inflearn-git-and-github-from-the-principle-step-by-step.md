## Lessons Learned
---
실무에서 바로 사용할 수준의 Git 활용법을 알려준다. 그래서 복잡하고 깊은 내용을 빼고 실용적인 내용으로만 구성되어있어서 좋다. 또한 복잡하게 느껴지는 GitHub의 개념을 그림으로 프레젠테이션해주어 더욱 쉽게 이해할 수 있는 듯 하다. 간단한 git bash 사용법도 알려주어서 다양한 환경에서도 사용할 수 있도록 도와준다.

Git의 본질에 대해 모르고 그냥 사용하던 입장에서 VCS로서의 Git의 특징과 장점을 알아가고 자세한 내부 로직을 이해하니 앞으로의 사용성이 더 좋아질 것이라 예상된다. 또한 Git 외의 다른 개발도구들도 더 알아보고 더 나아가 직접 개발하여 사용하고 싶은 생각도 든다.
